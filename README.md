
# Airbnb Project

An Airbnb-style full-stack web application built using Node.js, Express, MongoDB, and EJS.

## Features
- User Authentication
- MongoDB Session Storage
- MVC Architecture
- Dynamic Routing

## Tech Stack
- Node.js
- Express.js
- MongoDB Atlas
- EJS
- Tailwind CSS

## How to Run
1. Clone the repo
2. Run: npm install
3. Add your MongoDB URI in .env file
4. Run: npm start
